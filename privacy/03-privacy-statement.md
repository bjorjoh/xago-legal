## 2. Hva denne erklæringen gjelder

Denne personvernerklæringen gjelder for:
- **Besøkende på xago.no** og våre andre nettsteder
- **Sluttbrukere av Xago Dispatcher og Xago-apper** — det vil si ansatte hos våre kunder

Den gjelder **ikke** for personopplysninger som behandles inne i kundens Business Central-miljø. Det er kunden din arbeidsgiver som er behandlingsansvarlig for det — se din arbeidsgivers egne personvernrutiner.

## 3. Behandlingsgrunnlag og formål

Vi behandler personopplysninger på følgende grunnlag:

| Formål | Opplysninger | Grunnlag |
|---|---|---|
| Brukerautentisering og tilgangsstyring | Navn, e-post, telefonnummer (hashet), bruker-ID | Berettiget interesse / oppfyllelse av avtale |
| Levere og drifte Xago Dispatcher | Brukeridentitet, aktivitetslogg | Oppfyllelse av avtale |
| Feilsøking og support | Tekniske logger, brukeridentitet | Berettiget interesse |
| Kommunikasjon med kunder og kontaktpersoner | Navn, e-post, telefon, stillingstittel | Berettiget interesse / samtykke |
| Nettsideanalyse | IP-adresse, besøksdata (anonymisert der mulig) | Berettiget interesse / samtykke |

## 4. Hvilke opplysninger vi samler inn

### 4.1 Xago Dispatcher og BC-integrasjon

For sluttbrukere samler vi inn:
- Navn og e-postadresse (fra Microsoft-konto via Azure AD / Entra ID)
- Hashet telefonnummer (for varslinger — vi lagrer ikke klartekst)
- Brukerrolle og tilgangsnivå
- Aktivitetslogg knyttet til operasjoner i Dispatcher

Vi har **ikke** tilgang til innholdet i Business Central-databasen utover det som er nødvendig via GDAP-tilgang for support og implementering.

### 4.2 Nettside og markedsføring

- Kontaktskjemadata (navn, e-post, melding)
- Informasjonskapsler og analyse (se cookie-policy)

## 5. Deling med tredjeparter

Vi deler ikke personopplysninger med tredjeparter for markedsformål. Vi benytter følgende kategorier underdatabehandlere:

| Kategori | Eksempel | Formål |
|---|---|---|
| Skyinfrastruktur | Microsoft Azure | Drifting av Dispatcher-plattformen |
| Autentisering | Microsoft Entra ID | Brukerinnlogging |
| Kommunikasjon | [E-postleverandør] | Varsler og support-kommunikasjon |
| Analyse | [Analyseverktøy] | Nettsidetrafikk |

Fullstendig liste over underdatabehandlere er tilgjengelig på forespørsel eller via [xago.no/privacy].

Alle underdatabehandlere er underlagt databehandleravtaler og behandler kun opplysninger i henhold til våre instruksjoner.

## 6. Overføring utenfor EØS

Der vi benytter tjenester levert av Microsoft (Azure, Entra ID), lagres data primært i EØS. Der overføring til tredjeland skjer, er dette basert på EUs standardkontraktsbestemmelser (SCCs) eller tilsvarende overføringsmekanisme.

## 7. Lagringstid

Vi lagrer personopplysninger kun så lenge det er nødvendig for formålet:

- Brukerdata i Dispatcher: Slettes innen 90 dager etter avsluttet kundeforhold
- Support-logger: 12 måneder
- Kontaktskjema / e-post: Inntil 3 år fra siste kontakt
- Fakturaopplysninger: 5 år (bokføringsloven)

## 8. Dine rettigheter

Som registrert har du etter GDPR følgende rettigheter:

- **Innsyn** — du kan be om innsyn i hvilke opplysninger vi har om deg
- **Retting** — du kan kreve at feilaktige opplysninger rettes
- **Sletting** — du kan be om sletting ("retten til å bli glemt"), med forbehold for lovpålagte lagringsplikt
- **Begrensning** — du kan kreve begrenset behandling i visse tilfeller
- **Dataportabilitet** — du kan be om å få dine opplysninger utlevert i maskinlesbart format
- **Innsigelse** — du kan protestere mot behandling basert på berettiget interesse

For å utøve rettighetene dine, kontakt oss på [privacy@xago.no](mailto:privacy@xago.no). Vi svarer innen 30 dager.

Du har også rett til å klage til **Datatilsynet** (datatilsynet.no) dersom du mener behandlingen er i strid med personvernregelverket.
